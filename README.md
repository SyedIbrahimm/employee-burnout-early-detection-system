# 🧠 Employee Burnout Early Detection System

## Overview

Employee Burnout Early Detection System is a web-based application that helps identify employees who may be at risk of burnout based on workplace performance metrics.

## Features

- Employee Dashboard
- Burnout Risk Analysis
- Healthy / At Risk / Critical Classification
- Bonus & Salary Increment Recommendation
- Responsive User Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Tailwind CSS
