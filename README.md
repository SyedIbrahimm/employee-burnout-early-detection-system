# 🧠 Employee Burnout Early Detection System

## 📌 Overview

The Employee Burnout Early Detection System is a web-based application developed to identify employees who may be at risk of burnout using workplace performance metrics. The system analyzes employee data and classifies employees into Healthy, At Risk, and Critical categories. It also recommends bonuses and salary increments for high-performing employees.

---

## ✨ Features

- Employee Wellness Dashboard
- Burnout Risk Score Calculation
- Healthy / At Risk / Critical Classification
- Bonus & Salary Increment Recommendation
- Employee Details Dashboard
- Responsive User Interface
- Live Monitoring Dashboard
